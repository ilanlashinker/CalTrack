# 🥗 CalTrack

> Simple, offline-first calorie tracking with weight trends and activity logging.

אפליקציית ווב מלאה למעקב קלוריות, משקל ופעילות גופנית פועלת כולה בדפדפן, ללא שרת וללא התחברות.

---

## ✨ פיצ'רים עיקריים

### 🗓️ תפריט שבועי
- מעקב קלוריות יומי מחולק ל-4 ארוחות (בוקר, צהריים, ערב, נשנוש)
- סרגל התקדמות חזותי עם ציון צבעוני (ירוק / צהוב / אדום) לפי יעד קלורי
- השלמה אוטומטית (autocomplete) של מאכלים מהיסטוריה קודמת
- לחצני ניווט בין ימי השבוע

### 📅 לוח שנה אינטראקטיבי
- תצוגה חודשית עם נקודות צבע לפי ביצוע יומי
- לחיצה על יום עבר מציגה סיכום ארוחות בחלון מוקפץ

### ⚖️ מעקב משקל
- רישום משקל יומי עם גרף SVG מובנה
- חישוב ירידה כוללת וירידה שבועית
- גרף משולב משקל × ממוצע קלורי שבועי (תצוגת מגמה)

### 🏃 מעקב פעילות גופנית
- תמיכה ב-6 סוגי פעילות: הליכה, ריצה, אימון, שחייה, רכיבה, אחר
- חישוב הערכת שריפת קלוריות לפי נוסחת MET + משקל המשתמש
- מד רצף (streak) ימים פעילים רצופים

### 📉 גרעון קלורי
- חישוב TDEE (הוצאה קלורית יומית) לפי נוסחת **Mifflin-St Jeor**
- חישוב TDEE דינמי מבוסס היסטוריה (EMA על שינויי משקל × קלוריות בפועל)
- תצוגת גרעון / עודף יומי עם ציון חזותי

### 🔥 מאזן יומי
- חישוב קלוריות נטו: אכולות פחות שרופות
- תצוגה לפי ימי פעילות

### ⚙️ הגדרות
- שם משתמש, יעד קלורי יומי, משקל התחלתי
- נתוני TDEE: גיל, גובה, מין, רמת פעילות

### 💾 גיבוי ושחזור
- ייצוא כל הנתונים לקובץ **JSON**
- שחזור מקובץ גיבוי
- ייצוא לוג מפורט ל-**Excel** (3 גיליונות: פירוט, סיכום יומי, גרעון)

## 🛠️ טכנולוגיות

| טכנולוגיה | שימוש |
|---|---|
| HTML5 + CSS3 | מבנה ועיצוב (RTL מלא, ערבית/עברית) |
| Vanilla JavaScript | כל הלוגיקה, ללא framework |
| LocalStorage | שמירת כל הנתונים בדפדפן |
| SVG | גרפי משקל ומגמה מובנים |
| SpreadsheetML | ייצוא Excel ללא ספריות חיצוניות |

## 🚀 הפעלה

```bash
# אין צורך בהתקנה — פתח את הקובץ ישירות בדפדפן
open menu.html
```

או פשוט גרור את `menu.html` לחלון הדפדפן. **אין תלויות חיצוניות, אין npm, אין שרת.**

## 📊 מבנה הנתונים ב-LocalStorage

| מפתח | תוכן |
|---|---|
| `foodTrackerV2` | ארוחות לפי תאריך |
| `weightTrackerV1` | רשומות משקל יומיות |
| `activityTrackerV1` | פעילויות גופניות |
| `foodHistoryV1` | היסטוריית מאכלים לאוטוקומפליט |
| `settingsV1` | הגדרות משתמש |

---

## ⚠️ הגבלות ידועות

- הנתונים שמורים **רק בדפדפן הנוכחי** — לא מסונכרנים בין מכשירים
- חישוב שריפת קלוריות לפי MET הוא הערכה בלבד (שגיאה אפשרית של 30–50%)
- חישוב TDEE מדויק לפחות ±10%
